# 중고 거래&경매 플랫폼 개발
한이음 ICT 프로젝트를 통해 중고물품 거래와 중고물품 경매에 대한 웹을 만들어 보았다

## 주요 기능
중고 거래 및 경매 서비스 : 판매자는 본인의 물건을 중고 거래 혹은 경매에 등록하여 판매할 수 있으며, 구매자는 판매자가 등록한 물건을 구매할 수 있습니다.
